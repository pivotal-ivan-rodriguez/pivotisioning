#Pivotisioning v1.2

##Usage

```
* Archive your project (The way you are already use to: Menu->Product->Archive)
* Launch the Pivotisioning app
* Select the archive file (.xcarchive)
* Either select a destination folder or leave the "Same as Archive" checkbox on
* Enter a name for the IPA
* Select the provisioning profile that you used to sign the Archive
* Click on the "Create" button and thats it!
```
